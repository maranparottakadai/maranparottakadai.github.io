SAMPLE APPLICATION
Date : 24/07/2024
Aim:
To create a simple React component that displays a welcome message, centered on the screen,
with the text "SRMIST" in blue color.
Procedure:
1. Set Up the Project:
• Initialize a React project using Vite or create-react-app.
• Navigate to the project folder and open the App.js file.
2. Code the Component:
• Define a functional component App.
• Use inline styles to set up a flex container, centering the text both horizontally and
vertically in the viewport.
• Add a h1 tag that displays the welcome message and applies blue color to "SRMIST"
using inline styling.
3. Run the Application:
• Save the App.js file.
• Open a terminal and run npm start to start the development server.
• View the centered message in the browser.
Source Code:
function App() {
return (
<div style={{ display: 'flex', justifyContent: 'center', alignItems: 'center', height:
'100vh' }}>
<h1>
WELCOME TO <span style={{color:'blue'}}>SRMIST</span> MCA
</h1>
</div>
);
}
export default App;
